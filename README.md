# ns - NPM User Switcher

## Install
```sh
go install github.com/UnwrittenFun/ns
```

## Usage

To use, place your .npmrc files into `~/.ns` (or `C:/Users/USERNAME/.ns` on windows for example).
You must name them in the format `PROFILE_NAME.npmrc`

E.g.
```sh
~ ls -1
first_user.npmrc
other_user.npmrc
```

Then simply run
```sh
ns
```
